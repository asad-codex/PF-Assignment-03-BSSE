# Programming Fundamentals — Assignment 3

30 C++ programs covering arrays, 2D arrays, functions, function overloading, character arrays / strings, searching, and sorting.

## Programs

| File | Description |
|------|--------------|
| `AreaOverload.cpp` | Function overloading — area of square and rectangle |
| `ArraySum.cpp` | Calculate the sum of all scores in an array |
| `Average.cpp` | Calculate average of test scores in a 1D array |
| `BowlingAverage.cpp` | Calculate bowling average with division-by-zero check |
| `BubbleSortChars.cpp` | Sort characters alphabetically using bubble sort |
| `CopyWord.cpp` | Copy characters from one array to another |
| `CustomPower.cpp` | Custom power function without using `pow()` |
| `Display2DArray.cpp` | Store and display a 2D array (seating chart) |
| `EvenOddCount.cpp` | Count even and odd numbers in an array |
| `Factorial.cpp` | Calculate factorial of a number |
| `FullName.cpp` | Combine first name and last name into a full name |
| `GCD.cpp` | Find GCD of two numbers using Euclidean algorithm |
| `GetLength.cpp` | Get length of a character array using a loop |
| `LeapYear.cpp` | Check if a year is a leap year |
| `LinearSearch.cpp` | Linear search for a player's initial in a character array |
| `MinMax.cpp` | Find the lowest and highest score in an array |
| `PasswordCheck.cpp` | Verify password by comparing character arrays |
| `ReverseWord.cpp` | Reverse a word in place using a character array |
| `SecondMinMax.cpp` | Find the 2nd lowest and 2nd highest score in an array |

| `WordLength.cpp` | Calculate word length without using `strlen()` |
